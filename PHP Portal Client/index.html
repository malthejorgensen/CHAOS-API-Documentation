CHAOS.Portal PHP Client¶

The CHAOS.Portal PHP Client is a PHP library that facilitates easy interaction with a CHAOS.Portal endpoint.

The client wraps the CHAOS.Portal API in a PHP object with member functions for searching and getting objects.
